## Sample usage of \`[diepvries](https://github.com/PicnicSupermarket/diepvries)\` in Orchest

### Environment variables

Use the following environment variables to configure the pipeline

```SNOWFLAKE_USER
SNOWFLAKE_PASSWORD
SNOWFLAKE_WAREHOUSE
SNOWFLAKE_ACCOUNT```
